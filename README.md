 
# 🚗 Car Price Prediction

## 📌 Problem Statement

Predict the price of a car based on features like engine size, horsepower, and car dimensions.

## 📊 Dataset

* 205 rows, 26 columns
* Includes numerical and categorical features

## 🧠 Approach

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Correlation analysis for feature selection
* One-hot encoding for categorical variables
* Built Linear Regression model

## ⚙️ Model Performance

* Initial model: R² = 0.57
* Improved model: R² = 0.67

## 📈 Key Insights

* Engine size, horsepower, and curb weight strongly influence price
* Fuel efficiency features are negatively correlated with price

## 🛠️ Tools Used

* Python, Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
